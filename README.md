# Colossal Rigout

A modern, premium e-commerce fashion store built with Next.js 15, Tailwind CSS 4, and Firebase.

## Tech Stack

- **Framework:** Next.js 15 (App Router, TypeScript)
- **Styling:** Tailwind CSS 4
- **Database & Auth:** Firebase v12 (Firestore & Firebase Auth)
- **Storage:** Firebase Storage (category & product images)
- **Icons:** Lucide React
- **Animation:** Motion

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Dynamic product catalog with Firestore backend
- Admin dashboard with full CRUD operations
- Dynamic category management with image uploads
- Shopping cart, wishlist, and checkout flow
- Order tracking system
- Responsive design (mobile + desktop)
- Hero carousel management
- Promo code system
